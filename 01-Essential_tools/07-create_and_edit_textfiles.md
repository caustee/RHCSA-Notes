# Create and edit text files

## nano editor
```sh
# create file
nano file

# edit file
nano /etc/passwd
```

## vi and vim
```sh
# vim is vi improved

# create file
vi file  # by default enter in command mode

# use hjkl for navigate
# :q! quit unsaved changes
# :wq quit saving changes
# :!ls execute a command and return to editor

# use vimtutor for more
```
